# Contributing

Thank you for your interest in contributing to this project!

## Getting Started
1. Fork the repository
2. Clone your fork locally
3. Configure the database connection in `appsettings.json`
4. Set development secrets using .NET User Secrets
5. Apply database migrations
6. Run the project locally

## Branching Strategy
- Use feature branches: `feature/<short-description>`
- Keep commits small and focused

## Pull Requests
- Ensure the project builds successfully in Release mode
- Follow existing coding conventions
- Clearly describe what your change does and why it is needed

## Code Style
- Follow standard C# and ASP.NET Core best practices
- Keep controllers thin and business logic in services
